# Naikamu Mobile

## Build Naming

```
naikamu-{env}-{version}
```

Ex.

```
naikamu-stg-1.0.0-beta.4.1.0
```

For `.env` file:

```bash
# To untracked file:
git update-index --assume-unchanged [<file> ...]

# To undo and start tracking again:
git update-index --no-assume-unchanged [<file> ...]
```
